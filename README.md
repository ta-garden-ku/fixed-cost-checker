# 固定費削減診断 MVP

## ファイル構成

- `index.html`: LP、診断フォーム、結果画面、FAQ、構造化データ
- `styles.css`: スマホ対応デザイン、CTA、結果カード、危険度表示
- `script.js`: 入力管理、サブスク追加、診断計算、結果描画

## セットアップ手順

静的サイトとしてそのまま動きます。

1. `index.html` をブラウザで開く
2. 入力フォームを進める
3. Step 4で診断結果を確認する

公開URL:

- https://fixed-cost-checker.vercel.app/

Next.js化する場合は、このMVPのHTML構造を `app/page.tsx`、計算ロジックを `lib/calculator.ts`、入力UIを `components/` に分割する想定です。

## 計算ロジック

- 固定費合計 = サブスク以外の固定費 + サブスク合計
- サブスク合計は、固定費欄のサブスク概算合計と詳細リスト合計の大きい方を採用
- 年間固定費 = 固定費合計 × 12
- 固定費比率 = 固定費合計 ÷ 手取り収入 × 100
- スマホ代が8,000円以上なら30%を削減可能性として計上
- インターネット代が6,000円以上なら20%を計上
- 保険料が15,000円以上なら20%を計上
- 電気・ガス・水道が25,000円以上なら10%を計上
- ほぼ使っていないサブスクは80%を計上
- 契約を忘れていたサブスクは100%を計上
- 月に数回のサブスクは50%を計上
- 家計危険度は固定費比率、貯金率、未使用サブスク数から0〜100で算出

## 収益化導線

- 無料相談: 診断結果直後の最重要CTA
- LINE登録: 診断結果保存、家族共有、ステップ配信への導線
- プロ相談: 保険、通信、光熱費などの高単価案件へ接続
- おすすめサービス: アフィリエイトリンクの受け皿
- 有料PDFレポート: 診断結果を保存したいユーザー向け

## 追加した導線ページ

- `consultation.html`: 無料相談フォームの仮受付ページ
- `line.html`: LINE保存とステップ配信の説明ページ
- `recommended-services.html`: スマホ、保険、光熱費、回線などの仮アフィリエイト導線
- `paid-report.html`: 有料PDFレポート販売の仮ページ
- `thanks.html`: Googleフォーム送信後に案内する次のステップページ
- `lead.js`: 仮フォーム送信時の完了メッセージ表示
- `site-config.js`: LINE URL、アフィリエイトURLなどの差し替え設定
- `link-config.js`: 設定ファイルのURLをページ内リンクへ反映
- `tracking.js`: CTAクリックの仮ログをブラウザ内に保存
- `LINE_SETUP.md`: LINE公式アカウントの文面・ステップ配信・リッチメニュー案

本番化するときは、`consultation.html` のフォームをGoogleフォームやCRMに接続し、`line.html` のLINE URLを公式アカウントURLに差し替え、`recommended-services.html` の各リンクを提携先URLへ変更してください。

## 次にやること

1. LINE公式アカウントを作成
2. 友だち追加URLを取得
3. `site-config.js` の `lineOfficialUrl` を差し替え
4. `LINE_SETUP.md` のあいさつメッセージとステップ配信をLINE管理画面へ登録
5. Googleフォームの確認メッセージに `https://fixed-cost-checker.vercel.app/thanks` を入れる
6. `recommended-services.html` の仮アフィリエイトURLを本物に差し替え
7. Google Analyticsなどを入れ、`tracking.js` の仮ログを本番計測に置き換え

## CVR改善用コピー案

ファーストビュー案:

1. あなたの固定費、年間いくらムダになっているか知っていますか？
2. 毎月お金が残らない理由、固定費に隠れているかもしれません。
3. スマホ代・保険・サブスクの見直し余地を1分でチェック。

CTA案:

1. 無料で診断する
2. 1分で年間コストをチェック
3. 削減できる可能性を見る
4. 診断結果をLINEで保存する
5. 無料で見直し相談をする

危機感コピー方針:

- 誇大表現や断定は避ける
- 「可能性があります」「概算です」を入れる
- 月額ではなく年間・10年の差で見せる
- 不安だけで終わらせず、見直せば改善できる印象にする
