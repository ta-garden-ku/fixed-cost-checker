# 固定費削減診断 MVP

公開URL:

- https://fixed-cost-checker.vercel.app/

## 方針

このサイトは、運営者が個別返信しない前提の自己完結型サイトです。

目的:

- 診断で固定費の見直し余地を可視化する
- LINEで診断結果を保存してもらう
- 自動配信で見直し手順を届ける
- おすすめ見直しサービスやPDFレポートへ誘導する
- 将来的に広告、アフィリエイト、PDF販売で収益化する

手動対応が必要な無料相談フォームは停止します。

## ファイル構成

- `index.html`: LP、診断フォーム、結果画面、FAQ、構造化データ
- `styles.css`: スマホ対応デザイン、CTA、結果カード、危険度表示
- `script.js`: 入力管理、サブスク追加、診断計算、結果描画
- `line.html`: 診断結果コピー/LINE共有ページ
- `line-save.js`: 診断結果保存用テキスト生成
- `consultation.html`: 個別相談ではなく、自己完結型の見直しナビ
- `recommended-services.html`: スマホ、保険、光熱費、回線などの仮アフィリエイト導線
- `paid-report.html`: 有料PDFレポート販売の仮ページ
- `thanks.html`: 次のステップページ
- `site-config.js`: LINE URL、アフィリエイトURL、GA4 IDなどの設定
- `analytics.js`: GA4タグ読み込み
- `tracking.js`: CTAクリックイベント計測
- `LINE_SETUP.md`: LINE公式アカウントの自動運用メモ

## 収益化導線

- LINE登録: 診断結果保存、ステップ配信への導線
- おすすめサービス: アフィリエイトリンクの受け皿
- 有料PDFレポート: 診断結果を保存したいユーザー向け
- 広告: SEO流入が増えた後に導入

## 次にやること

1. Googleフォームへの相談導線をLINEやサイトから外す
2. LINEの応答メッセージを「個別返信なし」文面に変更
3. `recommended-services.html` の仮アフィリエイトURLを本物に差し替え
4. Google Analyticsで `cta_click` を確認
5. 5〜10人に触ってもらい、離脱箇所を見る

## 注意

保険・通信・電力等の契約判断はユーザー自身が行う前提です。
サイト側では断定的な助言を避け、一般的な確認順と比較導線を提示します。
