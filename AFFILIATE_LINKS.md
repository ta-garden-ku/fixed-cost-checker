# アフィリエイトリンク差し替えメモ

このサイトの収益リンクは `site-config.js` にまとめています。
ASPで広告URLを取得したら、該当するURLだけ差し替えてください。

## 差し替える場所

```js
affiliate: {
  sim: "格安SIM・スマホプランの広告URL",
  insurance: "保険見直し・保険比較の広告URL",
  energy: "電気・ガス切り替えの広告URL",
  internet: "ネット回線・Wi-Fiの広告URL",
  budget: "家計管理アプリの広告URL",
  creditCard: "クレジットカードの広告URL",
  securities: "証券口座の広告URL",
}
```

## 優先順位

1. 格安SIM・スマホプラン
2. 保険見直し・保険比較
3. 電気・ガス切り替え
4. ネット回線・Wi-Fi
5. 家計管理アプリ・クレカ・証券口座

## 運用方針

- 個別相談を受けず、診断結果から比較ページへ自然に流す
- 「解約すべき」ではなく「確認する」「整理する」という表現にする
- 子育て世帯には、教育費を削る訴求ではなく「教育費を守るために固定費を整える」と伝える
- ASPの規約に合わせて、広告表記やPR表記を必ず確認する
